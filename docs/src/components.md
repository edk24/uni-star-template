# 组件

## 原子化样式

原子化样式并非组件，是配置在 `uno.config.ts` 中的 `shortcuts` 快捷方式；

**用法**

```html
<button class="btn-empty">按钮</button>
```

| class 样式            | 说明                              |
| --------------------- | :--------------------------------- |
| `btn-empty`           | 去除 uniapp 默认按钮样式          |
